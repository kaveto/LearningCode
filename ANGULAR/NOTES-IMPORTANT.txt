Creating a New Project with Modules:
When you create a new Angular project, you wonâ€™t have any modules by default unless you specify otherwise.
To create a module-based app, use the --no-standalone flag when generating a new project:

ng new my-app --no-standalone

This will ensure that the familiar app.module.ts file is included in your project.
